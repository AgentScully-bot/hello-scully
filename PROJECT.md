# Project: hello-scully

## Overview
A simple Python CLI app that greets Agent Scully. Serves as the first test project to validate the full workflow: project setup → coding → testing → GitHub publish.

## Goals
- [ ] Working Python CLI that prints "Hello Scully!" with some personality
- [ ] Clean project structure (pyproject.toml, README, tests)
- [ ] Published to GitHub under AgentScully-bot
- [ ] CI pipeline with GitHub Actions (lint + test)

## Stack & Tools
- Language: Python 3.12
- Packaging: pyproject.toml (no setup.py)
- Testing: pytest
- Linting: ruff
- CI: GitHub Actions

## Constraints
- Keep it simple — this is a workflow test, not production software
- No external dependencies beyond dev tools (pytest, ruff)

- GitHub: AgentScully-bot/hello-scully
- Branch strategy: main only (simple project)
